Motd : Modification du message d'ouverture de console

From Deimos.fr / Bloc Notes Informatique
Jump to: navigation, search

Le message d'ouverture de votre console ne vous plait pas ? Je vous propose de la changer. Le fichier se trouve ici : 

/etc/motd

Ensuite, il existe pleins de petits logiciels pouvant faire des dessins et autre :

  • Cowsay : Permet de mettre du texte dans un petit dessin (par défaut une vache mais d'autres desssins sont dispos)
sudo apt-get install cowsay

Exemple: 

echo Serveur Toto|cowsay -f eyes
ls /usr/share/cowsay/cows
  • Boxes : Permet de mettre du texte dans des petits dessins (essentiellement pour mettre des commentaires dans des lignes de code).
sudo apt-get install boxes

Exemple: 

echo Acces on this server is stricly restricted | boxes -d peek
  • LinuxLogo : Pemet d'utiliser des bannières "linux" colorisées
sudo apt-get install linuxlogo

Exemple : 

linux_logo
  • FIGlet : Permet d'écrire du texte sous frorme de dessins ASCII.
sudo apt-get install figlet

Exemple : 

figlet -f small Access Restricted
showfigfonts|more
  • J'en ai marre, je ne veux plus rien en SSH ! Si vous ne désirez plus avoir de messages au boot via SSH, cette commande suffit :
touch ~/.hushlogin
