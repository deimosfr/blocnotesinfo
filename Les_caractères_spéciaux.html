1 Introduction

Le format d'encodage des caractères de spages HTML est par défaut : l'UTF-8 c'est à dire le format américain. Or ce dernier ne comporte pas nos caractères accentués latins.

Il est nécessaire, pour se conformer à cette norme d'encoder nos caractères spéciaux en un format compréhensible par cette norme. c'est pourquoi il existe deux codages : l'un ISO en format numérique et l'autre spécifique à HTML exprimé en langage naturel.

Un code ISO s'écrit ainsi : &#code;, alors qu'un code HTML s'écrit ainsi : &nom;.

2 Règles générales de codage

Les codes HTML sont des abréviations mnémotechniques (en langue anglaise) des lettres accentuées.

Syntaxe :

& lettre + abréviation ;

Exemple pour É :

Règle : & E + acute ;

Code réel : & Eacute ;
