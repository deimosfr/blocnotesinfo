1 Dépanner Outlook 2003

Essayez de lancer Outlook en utilisant cette commande : outlook /safe
Si Outlook se lance sans problème fermez l'application puis essayez de la relancer normalement.
Si vous avez le même problème commencez par supprimer les fichiers outcmd.dat et .srs (cela corrige des problèmes de lenteur).
Ces fichiers sont placés dans \Documents and Settings\Nom_Utilisateur\Application Data\Microsoft\Outlook . Voici une liste de fichiers pouvant engendrer des problèmes :

  • *.NK2 : nom d'utilisateur et cache des entrées permettant la fonctionnalité de suggestion des noms lors de la saisie des champs A, Cc et Cci.
  • *.SRS : paramètres d'envoi et de réception.
  • *.XML : volet de navigation.
  • *.FAV : barre d'Outlook ancienne version (ces fichiers ne sont pas utilisés par Outlook 2003).
  • Outcmd.dat : personnalisation de la barre d'outils.
  • Outlprnt : paramètres d'impression.
  • Fichiers VBA.

Ni vous n'obtenez pas de meilleurs résultats essayez de créer un nouveau profil. Par ailleurs, beaucoup de problèmes apparemment insolubles se résolvent de cette façon :
1) Cliquez sur Outils/Options...
2) Cliquez sur l'onglet Autre puis sur le bouton Options avancées...
3) Cliquez sur le bouton Gestionnaire de compléments puis décochez un à un les compléments installés jusqu'à localiser le coupable.

Vous pouvez également forcer Outlook à se lancer sans aucun complèment activé :
1) Dans le Registre ouvrez HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Office\Outlook\Addins.
2) Ouvrez les clés représentant les compléments installés.
3) Editez une valeur chaîne nommée loadbehavior.
4) Saisissez comme données de la valeur le chiffre 1 afin d'empêcher son chargement automatique à chaque démarrage d'Outlook (dans ce cas les données correspondent au chiffre 3).

Si rien de tout ceci n'a fonctionné cliquez sur le menu d'aide puis sur la commande Détecter et réparer.
Notez qu'un certain nombre de commutateurs de démarrage peut vous aider à identifier le composant d'Outlook qui est endommagé. Afin de, par exemple, démarrer Outlook sans activer l'affichage personnalisé saisissez cette commande : outlook /cleanviews

Reportez-vous en début de page afin de voir la liste de tous les commutateurs disponibles.

2 L'interface de messagerie a renvoyé une erreur inconnue. Si le problème persiste, redémarrez Outlook"

Si l'erreur survient quand vous voulez ajouter ou supprimer des contacts suivez cette procédure :
Exportez le fichier Outlook.pst :
1) Cliquez sur Fichier/Importer et exporter…
2) Sélectionnez l'option Exporter des données vers un autre fichier puis cliquez deux fois sur le bouton Suivant.
3) Cochez la case Inclure les sous-dossiers puis cliquez sur les boutons Suivant et Terminer.
4) Renommez le fichier Outlook.pst en ce que vous voulez.
Il se trouve dans \Documents and Settings\Nom_Utilisateur\Local Settings\Application Data\Microsoft\Outlook.
5) Relancez Outlook qui va créer un nouveau fichier .pst.
6) Importez selon la même procédure que précédemment le fichier .pst que vous avez sauvegardé.
Astuce communiquée par L. Thomas.

3 Outlook 2003 : "Je n'arrive plus à afficher le contenu de mes messages"

Ils s'affichent par contre impeccablement dans Outlook Express.
1) Lancez Internet Explorer.
2) Cliquez sur Outils/Options Internet puis sur le bouton Paramètres.
3) Dans la rubrique Dossier Temporary Internet files augmentez l'espace disque à utiliser pour les fichiers temporaires d'Internet Explorer.
Astuce communiquée par N. Berard.

4 "Interface non enregistrée"

Le problème peut se poser quand vous envoyez ou déplacez des mails.
Cliquez sur Démarrer/Exécuter puis saisissez : Regsvr32.exe %Windir%\System32\Ole32.dll

5 Impossible de récupérer la barre d'outil standard dans Outlook

Lancez Outlook en saisissant cette commande : outlook /ResetOutlookBar

6 "C:\Documents and Settings\User\Application Data\Microsoft\Macros complémentaire\c:\Program Files\Microsoft Office\Office\SBCMSYNC.DLL n'est pas un macro complémentaire d'Office valide"

Essayez tour à tour ces trois solutions :
1) Dans Outlook cliquez sur Outils/Options…/Autre puis sur le bouton Options avancées…
2) Cliquez sur le bouton Gestionnaire de compléments…
3) Décochez la case Outlook Sync puis relancez Outlook.
1) Renommez un fichier nommé Extend.dat présent dans \Documents and Settings\Nom_Utilisateur\Application Data\Microsoft\Outlook.
1) Dans l'éditeur du Registre supprimez cette entrée :
HKEY_CURRENT_USER\Software\Microsoft\Office\Outlook\Addins\SBCM.ADDIN.1.
Notez que si vous utilisez le Gestionnaire de clients pour petites entreprises, il vous suffit tout simplement de d'installer ce composant.

7 "Opération annulée à cause de restriction en vigueur sur cet ordinateur. Contacté votre administrateur systeme"

Le problème se pose quand vous cliquez sur un lien dans Outlook. 1) Dans Internet Explorer cliquez sur Outils/Options des dossiers...puis sélectionnez l'onglet Programmes. 2) Cliquez sur le bouton Rétablir les paramètres Web...

8 Les noms des dossiers sont en anglais

Vous allez vous retrouver avec des dossiers nommés Personnal folders, Inbox, etc.
Le problème se pose lors de l'installation d'un antivirus en version de langue anglaise.
Cliquez sur Démarrer/Exécuter puis saisissez : Outlook /ResetFolderNames

9 La première partie des mails ne s'imprime pas

Voici une solution de contournement :
1) Ouvrez le message qui pose problème.
2) Cliquez sur Edition/Modifier le message.
3) Cliquez sur Format/Texte brut ou Texte enrichi.
Sur les versions récentes d'Outlook se problème est résolu par l'installation du dernier service Pack.

10 "Erreur d'application"

Vous pouvez avoir également cette erreur : "L'instruction à "0x…" emploie l'adresse mémoire "0x…" La mémoire ne peut être "Read". Cliquez sur OK pour terminer le programme."
Dans Outlook, cliquez sur Outils/Options, puis sur l'onglet Autre. Cliquez sur les boutons Options avancées et Gestionnaire de compléments, puis décochez la case Dossiers personnels partagés.

11 Impossible d'ouvrir une pièce jointe dans Outlook

L'erreur exacte est : "Outlook a interdit l'accès aux pièces jointes suivantes, potentiellement dangereuses…" Cela peut s'appliquer à toutes les versions d'Outlook 2000 et ultérieures.
1) Cliquez sur Démarrer/Exécuter, puis saisissez : regedit.
2) Dans le Registre, ouvrez HKEY_CURRENT_USER\Software\Microsoft\Office\9.0 ou 10.0 ou 11.0\Outlook\Security.
3) Dans le volet de droite, créez une valeur chaîne nommée Level1Remove.
4) Éditez cette valeur, puis saisissez comme données de valeur les extensions des fichiers auxquels vous voulez accéder. Vous devez les séparer avec un point-virgule. Par exemple : .exe; .zip; .bat.
À l'inverse, il vous est possible d'ajouter des extensions de fichiers en paramétrant de la même façon une valeur chaîne nommée Level1Add.

12 Mes correspondants reçoivent une pièce jointe nommée Winmail.dat

Dans Outlook, cliquez sur Outils/Options, puis sur l'onglet Format de courrier. Dans la liste déroulante Format du, cliquez sur Texte brut.

13 Impossible d'utiliser le format de courrier Word pour mes mails

L'erreur exacte est : "Ce formulaire requiert l'utilisation de Word Mail comme éditeur de votre messagerie électronique, mais Word est occupé ou introuvable. Il sera ouvert dans l'éditeur Outlook à la place."
1) Après avoir effectué une sauvegarde de vos fichiers des dossiers personnels (.pst) dans Outlook, cliquez sur Démarrer/Exécuter, puis saisissez : regedit.
2) Dans l'éditeur du Registre, supprimez les clés suivantes :
HKEY_LOCAL_MACHINE\Software\Microsoft\Office\8.0
HKEY_LOCAL_MACHINE\Software\Microsoft\Office\9.0
HKEY_CURRENT_USER\Software\Microsoft\Office\8.0
HKEY_CURRENT_USER\Software\Microsoft\Office\9.0
3) Dans le module Ajout/Suppression de programmes du Panneau de configuration, sélectionnez votre version d'Office, puis cliquez sur le bouton Modifier.
4) Procédez à une réparation d’Office XP en cliquant sur le bouton radio Détecter et réparer les erreurs contenues dans l'installation de Office, puis sur Suivant.

14 Impossible d’ouvrir une pièce jointe Excel

Fermez Outlook, puis lancez Excel. Cliquez sur Outils/Options, puis sur l'onglet Général et décochez la case Ignorer les autres applications.

15 Je n'arrive pas à accéder aux paramètres d’impression

Lancez une recherche sur le fichier Outlprnt, puis supprimez-le.

16 "Impossible d'afficher le dossier. Microsoft Outlook n'a pas pu accéder à l'emplacement du dossier."

La suite du message d'erreur est : "Impossible de trouver le fichier .DLL pour le service d'informations. MAPI n'a pas pu charger le service d'informations PSTPRX.DLL.
Assurez-vous que le service est correctement installé et configuré." puis : "Des erreurs ont été détectées dans le fichier C:\Documents and Settings\Nom_Utilisateur\Local Settings\Application Data\Microsoft\Outlook\Outlook.pst. Fermez toutes les applications à extension messagerie, puis utilisez l'utilitaire Réparation de la boîte de réception".
1) Lancez une recherche sur ce nom de fichier : scanpst.exe.
2) Double-cliquez dessus.
3) Cliquez sur le bouton Parcourir...
4) Indiquez le nom de votre fichier .pst.
Le chemin par défaut est : \Documents and Settings\Nom_Utilisateur\Local Settings\Application Data\Microsoft\Outlook.
5) Cliquez sur le bouton Démarrer.
Il suffit après de lancer le processus de réparation.

17 Fichier Outlook.pst inaccessible - 0x80040116

Le problème peut se poser si le fichier .pst a une taille supérieure à 2 GB. Si l'exécution de l'outil scanpst.exe ne donne aucun résultat, suivez la procédure suivante :
1) Dans un moteur de recherche lancez une recherche sur ce nom de fichier : 2gb152.exe.
2) Téléchargez cet exécutable puis double-cliquez dessus afin que les fichiers soient décompressés dans le répertoire de votre choix.
3) Double-cliquez sur cet exécutable : PST2GB.exe.
4) Cliquez sur le bouton Browse puis sélectionnez le fichier .pst à tronquer.
Seuls les fichiers de plus de 2 GB apparaîtront...
5) Cliquez sur le bouton Create.
6) Indiquez le nom et l'emplacement de votre nouveau fichier .pst.
7) Dans la zone de texte Number of Megabytes to trim off the end:, inscrivez la plus petite valeur possible : de 15 MB à 20 MB par exemple.
En bref, il faut procéder par touches successives et, éventuellement, augmenter la valeur jusqu'à le processus soit couronné de succès.
8) Comme expliqué dans le paragraphe précédent, procédez à une réparation du fichier .pst en vous servant de l'utilitaire de réparation "Scanpst.exe".
Par la suite si vous parvenez à ouvrir ce fichier (Fichier/Ouvrir/Fichiers de dossiers personnels .pst), il sera listé dans la liste sous Dossiers personnels récupérés.

18 Supprimer un mot de passe protégeant un fichier .pst

Rappelons que pour créer un mot de passe protégeant les dossiers personnels il faut suivre cette procédure :
1) Cliquez sur Fichier/Gestion de fichiers de données.
2) Cliquez sur les boutons Paramètres... puis Modifier le mot de passe...
Vous avez besoin de deux utilitaires : scanpst.exe (qui se trouve dans \Program Files\Fichiers communs\System\MSMAPI\1036) et pst19upg.exe que vous pouvez télécharger à partir de cette adresse : http://www.computerhope.com/download/updates.htm.
Cette méthode ne fonctionnera pas avec les fichiers PST au format Unicode.
1) Copiez votre fichier outlook.pst dans un dossier temporaire avec les autres fichiers exécutables.
Il est placé dans \Documents and Settings\Nom_Utilisateur\Local Settings\Application Data\Microsoft\Outlook. Vous devez sélectionner le fichier portant le chiffre le plus important. Par exemple : Outlook5.pst. Pour plus de facilité j'ai copié les trois fichiers dans \Temp.
2) Exécutez scanpst.exe sur votre copie du fichier .pst.
3) Cliquez sur le bouton Parcourir afin de sélectionner la copie que vous avez effectuée du fichier .pst.
4) Cliquez sur les boutons Démarrer puis Réparer.
Bien que dans notre cas cela soit inutile, un fichier de sauvegarde nommé Outlook5.bak sera créé.
5) Ouvrez une fenêtre d'Invite de commandes puis placez-vous sur le répertoire temporaire.
6) Saisissez ceci : pst19upg.exe -x outlook5.pst
Un fichier portant l'extension .psx sera créé.
7) Renommez le fichier .pst en simplement changeant son extension : Outlook5.old
8) Saisissez maintenant cette commande : pst19upg.exe -i outlook5.psx
Un nouveau fichier portant l'extension .pst sera créé.
9) Exécutez à nouveau l'utilitaire scanpst.exe en sélectionnant ce fichier.
10) Relancez Outlook puis connectez-vous sur un profil différent.
11) Cliquez sur Fichier/Ouvrir/Fichiers de données Outlook puis sélectionnez le fichier que vous venez de réparer.
Si tout se passe normalement vous n'avez plus qu'à remplacer l'ancien fichier par le nouveau puis à vous connecter sur votre profil.
