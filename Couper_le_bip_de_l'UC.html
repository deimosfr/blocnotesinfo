1 Introduction

Que c'est fatiguant d'entendre BIP à chaque complétions impossibles ou errors lorsque l'on est dans un shell !

2 Solutions

2.1 A la volée

Tout simplement, depuis une interface graphique, lancez cette commande :

xset 
xset -b

Voilà donc la solution pour stoper le biiiiiiiiiiiiiiip :-)

2.2 Définitif

Vous souhaitez une solution sur le long terme ! Rajoutez cette ligne dans le fichier /etc/modprobe.d/blacklist :

/etc/modprobe.d/blacklist 
blacklist pcspkr

Et là tous les sons (allumage, extinction...) de l'UC seront désactivés ! Le bonheur !
