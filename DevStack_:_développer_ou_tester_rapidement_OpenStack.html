1 Introduction

Si vous voulez rapidement tester OpenStack[1], que ce soit pour jouer ou développer dessus, DevStack[2] est aujourd'hui la méthode la plus rapide.

Je suis parti sur une base d'Ubuntu Server 12.04 car cette distribution fait partie des versions recommandées par DevStack. Le but est donc ici d'avoir une VM avec tous les services d'OpenStack installés et fonctionnels.

WARNING
N'utilisez pas DevStack en production !!!

2 Installation

Pour la mise en place de DevStack, nous allons avoir besoin de git :

aptitude 
aptitude install git git-core

Nous allons donc maintenant récupérer DevStack dans sa version actuelle, puis brancher sur la version souhaitée (Grizzly) :

git 
git clone git://github.com/openstack-dev/devstack.git
cd devstack
git checkout stable/grizzly

Puis lancez l'installation et renseignez tous les mots de passe demandés :

./stack.sh 
./stack.sh

3 Utilisation

Pour lancer une stack de développement :

./stack.sh 
./stack.sh

Lorsque l'installation est terminée, vous pouvez accéder aux différents services comme ceci :

  • Horizon: http://serveur/
  • Keystone: http://serveur:5000/v2.0/

Les utilisateurs par défaut sont admin et demo

Pour plus d'inforamtions, consultez la page GitHub[3] de DevStack.
