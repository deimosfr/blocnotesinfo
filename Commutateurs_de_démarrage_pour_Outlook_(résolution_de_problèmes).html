Commutateurs de démarrage pour Outlook (résolution de problèmes)

From Deimos.fr / Bloc Notes Informatique
Jump to: navigation, search

Voici un ensemble de commutateurs vous permettant de paramétrer l'ouverture d'Outlook 2000 et ultérieur. Concernant les commutateurs disponibles sous Outlook 2003 servez-vous de l'aide en tapant cette recherche : commutateurs de ligne de commande.

Vous pouvez créer un nouveau raccourci ou modifier un raccourci existant. La syntaxe sera, par exemple : "C:\Program Files\Microsoft Office\Office10\Outlook.exe" /recycle

Vous obtiendrez plus de précisions à cette adresse web : http://office.microsoft.com/fr-ca/assistance/HP010031101036.aspx.

  • /a "chemin:\nom de fichier" : démarre Outlook avec un nouveau message auquel est joint le fichier spécifié.
  • /autorun Nom_Macro : exécute la macro spécifiée.
  • /c ipm.X : crée un nouvel item avec la catégorie de message spécifiée ci-dessous :

1) ipm.note : nouvel email. 2) ipm.post : nouvelle discussion. 3) ipm.appointment : nouveau rendez-vous. 4) ipm.task : nouvelle tâche. 5) ipm.contact : nouveau contact. 6) ipm.activity : nouvelle entrée dans le journal. 7) ipm.stickynote : nouveau Post-it.

  • /CheckClient : demander l'identification du responsable par défaut du courrier électronique, des news et des contacts. Une manière comme une autre de déclarer Outlook comme votre messagerie par défaut.
  • /CleanClientrules : démarre Outlook en supprimant les règles basées sur le client. À partir de Outlook 2003.
  • /CleanDmRecords : supprime les enregistrements de connexion enregistrés. À partir de Outlook 2003.
  • /CleanFreebusy : nettoie et régénère les informations de disponibilité. Ne fonctionne que lorsque vous vous connectez à un serveur Microsoft Exchange.
  • /CleanProfile : supprime les clés de profil non valides et recrée les entrées du Registre. À partir de Outlook 2003.
  • /CleanPst : démarre Outlook avec un fichier de dossiers personnels (.pst) purgé. À partir de Outlook 2003.
  • /CleanReminders : nettoie et régénère les mémorandums.
  • /CleanRules : supprime les règles basées sur le client et sur le serveur. À partir de Outlook 2003.
  • /CleanSniff : supprime les doublons de messages de rappel. À partir de Outlook 2003.
  • /CleanSubscriptions : supprime les messages d'abonnement. À partir de Outlook 2003.
  • /CleanViews : restaure les préférences par défaut. Tous les affichages personnalisés seront supprimés.
  • /Designer : empêche Outlook de vérifier s'il doit être le client par défaut. À partir de Outlook 2003.
  • /Explorer : ouvre la nouvelle fenêtre avec la barre de liens activée. À partir de Outlook 2003.
  • /f Nom_Du_Message : ouvre le fichier de message spécifié (.msg) ou la recherche enregistrée (.oss).
  • /FirstRun : démarre Outlook comme s'il s'agissait de la première exécution. À partir de Outlook 2003.
  • /Folder : ouvre Outlook sans la barre Outlook et la liste des dossiers.
  • /ImportPrf Nom_Fichier_Prf : ouvre Outlook en important le profil MAPI défini (*.prf). À partir de Outlook 2003.
  • /NoCustomize : démarre Outlook sans charger le fichier outcmd.dat (barres d'outils personnalisées) et les fichiers *.fav. À partir de Outlook 2003.
  • /NoExtensions : démarre Outlook avec les extensions désactivées (dans le Gestionnaire de compléments). À partir de Outlook 2003.
  • /NoPollMail : démarre Outlook sans vérifier le courrier électronique lors du démarrage. À partir de Outlook 2003.
  • /NoPreview : ouvre Outlook sans le volet de visualisation et l'option correspondante.
  • /Profile Nom_Du_Profil : lance un profil spécifique.
  • /Profiles : affiche la boîte de dialogue de choix d'un profil.
  • /Recycle : empêche de lancer Outlook dans une nouvelle fenêtre s'il est déjà ouvert par un autre utilisateur.
  • /regserver : réinscrit les clés de registre spécifiques à Outlook.
  • /ResetFolderNames : redéfinit les noms de dossier par défaut (tels que Boîte de réception ou Éléments envoyés). À partir de Outlook 2003.
  • /ResetFolders : restaure les dossiers manquants de la boîte de réception par défaut.
  • /ResetNavPane : efface et recréé le volet de navigation pour le profil en cours. À partir de Outlook 2003.
  • /ResetOutlookBar : restaure la barre par défaut d'Outlook.
  • /s Nom_De_Fichier : lance un raccourci spécifique (fichier .fav).
  • /safe : démarre Outlook sans extensions, volet de lecture, ni personnalisation de barre d'outils. À partir de Outlook 2003.
  • /safe:1 : démarre Outlook avec le volet de lecture désactivé. À partir de Outlook 2003.
  • /safe:2 : démarre Outlook sans vérifier le courrier électronique lors du démarrage. À partir de Outlook 2003.
  • /safe:3 : démarre Outlook avec les extensions désactivées. À partir de Outlook 2003.
  • /safe:4 : démarre Outlook sans charger Outcmd.dat (barres d'outils personnalisées) et le fichier *.fav. À partir de Outlook 2003.
  • /Select Nom_Répertoire : affiche le répertoire spécifié.
  • /sniff : force la détection de nouvelles demandes de réunion dans la Boîte de réception, puis les ajoute au calendrier. À partir de Outlook 2003.
  • /unregserver : supprime toutes les clés de registre et les associations de fichiers pour Outlook.
