1 Sauvegarder ses dossiers personnels

Cette fonctionnalité suplémentaire se télécharge à cette adresse :
http://www.microsoft.com/downloads/details.aspx?FamilyId=8B081F3A-B7D0-4B16-B8AF-5A6322F4FD01&displaylang=en. Double-cliquez sur le fichier pfbackup.exe pour lancer l'installation proprement dite. Une cela terminé, ouvrez puis fermez Outlook. Une fenêtre nommée Outlook Personal Folders backup lancera immédiatement l'assistant de sauvegardes de vos dossiers Outlook.

2 Rôles des fichiers respectifs dans Outlook 2003

.pst : fichiers de dossiers personnels.
.pab : carnet d'adresses personnel.
.ost : dossiers hors connexion.
.msnpst : dossiers de connexion MSN.
.fav : fichiers de raccourcis de la barre d'outils.
.rwz : fichiers de règles.
.nk2 : fichiers de cache servant à la fonction de suggestion des noms.
.htm, .txt et .rtf : fichiers de signature automatique au format HTML, texte brut et RTF.

Notez que si vous utilisez Microsoft Word comem éditeur par défaut les signatures sont stockées dans le fichier de modèle Normal.dot.

OutlPrnt : paramètres d'impression.
Outcmd.dat : paramètres de la barre d'outils.
Views.dat : personnalisation des dossiers système.
VbaProject.form : programmes macro et VBA.
.xml : personnalisation de la barre de navigation.
.srs : paramètres d'envoi et de réception.
.htm : papiers à lettres.
.dic : fichiers des dictionnaires.
extend.dat : compléments installés.

3 Autres éléments de sauvegarde

Les catégories sont stockées dans cette clé du Registre : HKEY_CURRENT_USER\Software\Microsoft\Office\11.0\Outlook\Categories.
Les préférences dans celle-ci : HKEY_CURRENT_USER\Software\Microsoft\Office\11.0\Outlook\Preferences.

4 Copier la barre et les raccourcis présents dans le volet de navigation d'Outlook

Il vous suffit de copier ce fichier : Nom_Utilisateur.xml placé dans \Documents and Settings\Nom_Utilisateur\Application Data\Microsoft\Outlook. Renommez simplement le fichier en fonction du nom donné au profil.

Notez qu'il est préférable si vous possédez Outlook 2003 de supprimer les fichiers portant l'extension .fav.
